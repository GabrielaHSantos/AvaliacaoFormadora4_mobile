# Projeto Contatos

## Objetivo
Esta aplicação tem como objetivo demonstrar os conceitos básicos de desenvolvimento mobile com Ionic e Firebase, incluindo:
- Controle de versão com Git
- Criação de componentes e páginas
- Consumo de API RESTful
- CRUD com Firebase
- Geração de build para produção

## Tecnologias Utilizadas
- Ionic
- Angular
- Firebase
- Git
